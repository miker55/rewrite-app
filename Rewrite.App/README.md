# Rewrite

Rewrite is a simple mobile app that helps you improve messages before sending them.

Paste or type a message, choose a tone, and Rewrite will instantly generate a better version that you can edit, copy, or share.

The goal is speed: fix awkward messages in seconds.

---

## Features

- Rewrite messages with different tones
- Edit the improved message before sending
- Copy results to the clipboard
- Share directly to other apps
- Simple and fast mobile interface

Tone options include:

- **Professional**
- **Polite**
- **Friendly**
- **Shorter**

---

## Example

Input:

> hey just checking if you saw my email and if we are still on for tomorrow

Output:

> Just checking to see if you received my previous email and whether we are still scheduled for tomorrow.

## Development

### Requirements

- Visual Studio 2022 / 2024
- .NET MAUI
- Xcode (for iOS builds)
- Android SDK (for Android builds)

### Running the app

1. Clone the repository

```bash
git clone https://github.com/yourusername/rewrite-app.git
```

2. Open the solution in Visual Studio.

3. Select a target platform (iOS Simulator or Android Emulator).

4. Run the project.
